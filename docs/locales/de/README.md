## Hinweis zur Software-Sprache

Diese Übersetzung betrifft nur die Dokumentation. Die Benutzeroberfläche der Software selbst wird unabhängig übersetzt — sehen Sie in der laufenden Anwendung in **Einstellungen → Sprachen** nach, um deutsche UI-Beschriftungen zu aktivieren.
