## Contribuir

Las contribuciones a la documentación en español son bienvenidas — incluso traducciones parciales. Vea [CONTRIBUTING-TRANSLATIONS.md](../CONTRIBUTING-TRANSLATIONS.md) (inglés) para las convenciones.

Prioridades de traducción:

1. **GLOSSARY.md** — la consistencia terminológica importa más que la cobertura
2. **FAQ.md** — referencia diaria más consultada
3. **INSTALLATION-CHECKLIST.md** — para administradores de sistemas

## Nota sobre el idioma del software

Esta traducción se refiere únicamente a la documentación. La interfaz de usuario del software se traduce por separado — en la aplicación, vea **Configuración → Idiomas** para activar las etiquetas en español.
