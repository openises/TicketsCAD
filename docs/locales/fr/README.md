## Contribuer

Les contributions à la documentation francophone sont les bienvenues — y compris les traductions partielles. Voir [CONTRIBUTING-TRANSLATIONS.md](../CONTRIBUTING-TRANSLATIONS.md) (anglais) pour les conventions.

Priorités de traduction :

1. **GLOSSARY.md** — la cohérence de la terminologie compte plus que la couverture
2. **FAQ.md** — référence quotidienne la plus consultée
3. **INSTALLATION-CHECKLIST.md** — pour les administrateurs systèmes

## Note sur la langue du logiciel

Cette traduction concerne uniquement la documentation. L'interface utilisateur du logiciel est traduite séparément — dans l'application, voir **Paramètres → Langues** pour activer les libellés français.
