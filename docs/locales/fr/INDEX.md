## Organisation de cette documentation

Nous suivons le cadre [Diátaxis](https://diataxis.fr) : quatre types de documentation, chacun avec un rôle distinct.

| Type | Objectif |
|---|---|
| **Tutoriels** | Apprendre par la pratique |
| **Guides pratiques** | Résoudre un problème spécifique |
| **Référence** | Consulter une information |
| **Explication** | Comprendre le contexte |
